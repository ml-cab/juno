# Funding

Juno is maintained by a small team. Financial support allows maintainers to
dedicate time to the project, fund CI infrastructure, and sustain long-term
development.

---

## Donation Channels

### GitHub Sponsors

The primary donation channel for individual developers and companies.

    https://github.com/sponsors/ml-cab

GitHub Sponsors supports one-time and recurring contributions. GitHub waives
platform fees for open-source sponsored accounts (verify current terms at
github.com/sponsors).

### Open Collective

Open Collective provides transparent public accounting — all income and expenses
are visible to anyone. Suitable for teams and companies that require invoice
documentation for accounting purposes.

    https://opencollective.com/juno-ml

One-time and recurring tiers available. Open Collective charges a platform fee
(currently 8% host fee + 5% Open Collective fee; verify current rates before
contributing).

### Direct Bank Transfer (Corporate)

For companies preferring a direct transfer without platform fees, contact
dev@ml.cab for wire transfer details. A formal invoice can be issued on request.

---

## How Funds Are Used

All expenditure via Open Collective is publicly itemised. Typical uses:

- Maintainer stipends for development time on core engine work.
- CI infrastructure costs (GPU-enabled build runners for CUDA/ROCm test coverage).
- Security audit engagements.
- Legal review of licenses and contributor agreements.
- Conference travel for project representation.

target programmes:

### Grant Readiness Requirements

To be eligible for most grants, the project maintains:

- Public source repository with Apache 2.0 license verified by REUSE or SPDX tooling.
- Contributor License Agreement on file (see [docs/CLA.md](docs/CLA.md)).
- Public financial accounting via Open Collective or equivalent.
- A governance document describing decision-making and maintainer succession
  (see [docs/GOVERNANCE.md](GOVERNANCE.md)).
- A security policy with a coordinated disclosure process (see [SECURITY.md](../SECURITY.md)).
- A code of conduct for contributors.

### To Propose a Joint Grant Application

If you represent a research institution or NGO and are interested in a joint
grant application involving Juno, contact dev@ml.cab with the subject line
"Grant Partnership Inquiry." Include the funding programme name, deadline, and
a brief description of the proposed scope.
